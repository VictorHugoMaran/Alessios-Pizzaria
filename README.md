# 🍕 Alessio's Pizzaria

Projeto de site para uma pizzaria fictícia chamado **Alessio's**, desenvolvido com foco em prática de **HTML e CSS** e criação de interfaces modernas para restaurantes.

O site apresenta uma landing page com destaque para o cardápio, navegação simples e um design visual voltado para experiência do usuário.

## 🚀 Tecnologias utilizadas


- HTML5
- CSS3
- Layout responsivo
- Boas práticas de estruturação de páginas web

## 📋 Funcionalidades

- Página inicial com destaque visual da pizzaria
- Navegação simples e intuitiva
- Seção de cardápio
- Página sobre a pizzaria
- Página de contato

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- Estruturação de páginas web
- Criação de layouts modernos
- Organização de componentes de interface
